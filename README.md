# Notch지 말아요 - 단어암기 앱(Global vocabulary)

## 스토어 링크
심사 끝나는 대로 추가 예정

## 기술스택 
이후에도 계속 추가될 예정이나, 현재 기술스택입니다.

언어 : kotlin, DI : Hilt

아키텍처 : MVVM
          
compose, coroutine, flow, hilt, room, admob, firebase analytics, firebase crashlytics


## 내용
처음에 개인적으로 영어단어 암기용으로 유틸앱을 만들어서 사용 중에 업그레이드하여 포트폴리오겸 상용화(?)를 목표로 만들었습니다.
영어, 한국어, 일본어, 중국어, 스페인어 5개 국어를 단어를 저장할 수 있어 외국인들도 한국어 또는 그 외 나라의 단어를 외울 수 있도록 구현했습니다.

## 화면 소개 

## 홈
주간 목표로 몇개의 단어를 외울것인지 목표를 설정할 수 있으며, 
오늘 저장한 단어 리스트들이 노출됩니다.
저장한 단어의 종류들을 기준으로 문제를 낼 수 있습니다. (ex. 영어, 한국어, 일본어 저장했을때 영어만 문제를 낸다거나 전체를 내는 방식)
또한 입력모드와 손글씨 모드로 나뉘어져있어 입력모드는 앱에서 정답을 맞추는 형태이며, 손글씨 모드는 앱이 아닌 직접 답을 공책 등에 쓰며 외울 수 있도록 
2가지의 모드를 만들었습니다.
또한 입력모드에서 자주 틀린 단어들을 모아 따로 문제를 풀 수도 있습니다.

## 단어장
단어를 저장하기 위해 단어장을 먼저 추가하며 언어의 카테고리를 설정합니다.
이후 단어장 안에서 단어들을 추가할 수 있으며 내가 저장한 단어에 한해 검색 및 수정, 삭제도 가능합니다.

## 캘린더
캘린더에서 내가 단어를 저장한 날이 표시되며, 해당 날짜를 클릭시 저장한 단어들의 리스트들을 보여줍니다.

## 차트
최근 1주일간 저장한 단어의 수에 대한 차트와 
입력모드로 문제를 푼 경우 정답률에 대한 차트를 보여줍니다.
